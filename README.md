Repository for ITCS6162 programming assignment for recommender systems with IMDB 100k dataset.
Movie_Recommendation.ipynb contains the following:
- Data processing and cleaning for users, ratings, and movies
- Implementation of user-user collaborative filtering
- Implementatino of item-item collaborative filtering
- Implementation of the weighted pixie algorithm for recommendation using weighted random walks

The movie recommendation notebook contains code for data exploration and recommender system implementations, while the two pdf files contain an explanation of pixie-inspired graph-based recommender systems and a project report.
